CSPP_RT is an addon for the CSPP_Core package of the CS++-Project. 

It contains an independent set of ancestor and concrete classes for Actor Framework based RT applications

Refer to https://github.com/HB-GSI/CSPP for CS++ project overview, details and documentation.

LabVIEW 2014 is the currently used development environment.
